# AES Research Center Soil Map

Interactive soil maps for Colorado State University Agricultural Experiment Station (AES) research centers, built with SSURGO data.

## Overview
- 10 AES research stations across Colorado
- SSURGO soil data served via ArcGIS Online Feature Server
- Soil profile visualization powered by NRCS OSD Munsell color data

## Stack
- ArcGIS JS API 4.28
- Tailwind CSS
- Hosted on GitHub / Cloudflare
