Data source & averaging method
  • Source: Values are read from CSU AES soils Feature Layers (ArcGIS FeatureServer). Layers are built from USDA-NRCS SSURGO geopackage data, sampled at 0-6" depth. Citation: Soil Survey Staff, Natural Resources Conservation Service, United States Department of Agriculture. Web Soil Survey. Available online at http://websoilsurvey.nrcs.usda.gov/. Accessed 7/15/2025.
  • Scope: Features within the station boundary only.
  • Averaging: Area-weighted by each feature's area field (e.g., shape_Area, EPSG:26913 / UTM Zone 13N m²). For every attribute we compute ∑(value × area) ÷ ∑(area) using only features with a value for that attribute.
  • Units: Sand/Silt/Clay/OM and CaCO₃ are %, pH is unitless, EC is dS/m (if populated), BD is g/cm³ (if populated).
  • Projection: Data uses EPSG:26913 (NAD83 / UTM Zone 13N) for accurate area calculations in Colorado.
  • Maps: Interactive maps show real-time data with MUSYM labels and can be configured with custom symbology.
