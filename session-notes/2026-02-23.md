## Lessons Learned
- **`align-self: flex-end` changes meaning when flex direction changes** â€” was pushing OM gauge to the right in column layout instead of to the bottom as intended in row layout
- **`justify-content: space-between` with padding offset** â€” 2px padding on tick labels but not on the gradient bar caused misalignment; removing padding fixed it
- **Tick label alignment with gradients** â€” `flex: 1; text-align: center` on each tick span gives more reliable alignment than `space-between` with bare spans of varying text width
- **Programmatic `goTo()` bypasses input event blockers** â€” disabling zoom via `stopPropagation()` on user input events doesn't affect JS-triggered zoom calls
