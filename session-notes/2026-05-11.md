#### CoAgMET soil moisture
- Data served from `/soil/soilmoisture.json` — local file in repo to avoid CORS block
- GitHub Action (`.github/workflows/update-soil-moisture.yml`) fetches `coagmet.colostate.edu/data/daily/soilmoisture.json?to=now&span=30` daily at 0600 UTC (midnight MT) and commits updated file
- Only OTH01 (Olathe) is actively reporting live VWC from CoAgMET
- Sensor depths: 4" and 20"; fields vwc4, vwc24, st4, st24
- VWC colors: #32A5E2 (4"), #164863 (20"); Temp colors: #D35400 (4"), #800B04 (20"), dashed lines

#### RAWS soil moisture
- Uses NIFC DATA4 endpoint: `https://weather.nifc.gov/ords/prd/wx/station/dataNesdisidRange/{start}/{end}/{NesdisId}`
- NESS ID captured automatically from the NIFC baseline API (`s['NESS ID']`) during network load and stored in `rawsNesdisMap`
- Date range constructed dynamically in JS (`fmtNifcDate()` → `YYYYMMDDHHmmss`)
- Sensor depths: 2", 8", 20"; fields SMF/SMF2/SMF3 (VWC, FRAC×100), STF/STF2/STF3 (temp °F)
- VWC colors: #32A5E2 (2"), #2986B8 dashed (8"), #164863 (20"); Temp: #D35400, #B03A00, #800B04 dashed
- Datasets with all-null values filtered out automatically
